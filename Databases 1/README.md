# Databases 1 UACS course


## Course outline/ description:
This course is designed for software engineers, database engineers, and programmers, providing foundational knowledge in managing relational database systems. It covers essential concepts of database design, with a focus on query processing and optimization, as well as efficient data retrieval.

Students will learn to structure and manage data effectively within relational databases and perform various database management tasks. Additionally, the course aims to develop the skills required to build and manage different types of database applications.

Key topics include relational database design, query optimization, and transaction processing. Other areas of focus include an introduction to database management systems (DBMS), relational models, SQL data definitions, entityrelationship (ER) modeling, UML and ODL, relational algebra, and advanced SQL concepts. Students will also learn about constraints, triggers, transactions, views, indexes, and functional dependencies. The course will address normalization techniques, dependency inheritance, database repair processes, Codd’s 12 rules, and essential aspects of database security and authorization.



## Learning objectives:

After the completion of the course, the students should gain the knowledge and understanding of:

* Explain the fundamental concepts of database management systems (DBMS) and the relational model.

* Design and implement efficient relational database structures using entity-relationship (ER) modeling and normalization techniques.

* Write SQL queries to retrieve, manipulate, and manage data, including the use of advanced SQL features such as joins, subqueries, triggers, and stored procedures.

* Analyze and optimize database performance through indexing, query optimization, and execution plans.

* Manage database transactions and implement concurrency control to ensure data consistency and integrity.

* Perform essential database maintenance tasks, including backup, recovery, and data security measures.

* Apply authentication, authorization, and encryption techniques to protect database systems and ensure compliance with data privacy regulations.

* Develop practical database-driven applications and solutions for real-world scenarios.

* Critically evaluate different database design strategies and make informed decisions to improve performance, scalability, and security.

* Collaborate effectively on database-related projects, presenting and documenting solutions clearly and professionally.

* Demonstrate lifelong learning skills by exploring emerging trends and advancements in database technologies.


## Learning outcomes

By the end of this module, students will achieve the following learning outcomes:

* Knowledge: Students will develop a comprehensive understanding of database systems, including core concepts such as relational models, SQL, data normalization, indexing, and transaction management. They will become familiar with key principles of database design, query optimization, and data security, as well as the use of constraints, triggers, views, and functional dependencies.

* Skills: Students will acquire practical skills in designing, creating, and managing relational databases. They will learn to write efficient SQL queries for data retrieval and manipulation, optimize database performance, and implement transaction management techniques. Through hands-on tasks and projects, students will apply these skills to solve real-world database problems and develop database-driven applications.

* Competences: Students will be able to critically evaluate various database design strategies and make informed decisions to improve database performance, integrity, and security. They will demonstrate competence in creating well-structured databases by integrating theoretical knowledge with practical implementation. Furthermore, students will enhance their communication and teamwork skills by presenting their work in projects, writing detailed reports, and participating in peer reviews. Through independent project work, they will also develop the ability to engage in lifelong learning and prepare for future academic or professional challenges.
