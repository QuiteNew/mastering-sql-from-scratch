This project was created for bonus points and focuses on the rich 50-year evolution of Structured Query Language (SQL). The primary purpose of this folder is to host a couple of presentation scripts designed for a clear and well-paced 5-minute speech. Instead of focusing on the technical setup of a repository, this project serves as a straightforward educational guide to understanding how a single database language survived decades of technological shifts to remain the dominant standard today.



In essence, the crux of the project is focused on the radical innovation of data management which occurred way back in 1970 when the revolutionary Relational Model was invented by Dr. Edgar F. Codd. He managed to revolutionize the concept of complex data networks and introduced simple and neat tables. Another important aspect which is discussed is the history of the practical implementation of the language which was carried out by the IBM researchers Donald Chamberlin and Raymond Boyce, who wanted to develop a data language that looks like regular English.



In addition, this project provides information on the amazing flexibility of SQL in the 21st century. Even though many people predicted that the "Big Data" and "NoSQL" era will render SQL useless, it managed to adapt by adding native XML and JSON web data support. Nowadays, it is used in huge cloud databases like Snowflake and BigQuery and continues to be an integral foundation skill set in modern technology stacks, even in mobile applications, finance, and Python-based artificial intelligence systems. In order to make the presentation clear, the scripts of this folder are arranged per slide.
