\# Microsoft Certified: Azure Data Fundamentals

Welcome to the first step of my data journey. This repository serves as my personal notebook and tracking dashboard as I prepare for the Microsoft Certified: Azure Data Fundamentals exam. I am starting here because data is the backbone of modern technology, and this certification is my launchpad into learning how data is stored, processed, and secured.



The goal is to build a rock-solid foundation in core data concepts and see how they come to life within Microsoft Azure. I am using this space to document my notes and key takeaways so that I can track my progression and look back on how I started.



To prepare for this exam, I am following a structured study plan using a mix of great community resources. I am learning the foundational concepts through Scott Duffy's DP-900 course, testing my knowledge with the official practice tests from the Microsoft website alongside other mock exams, and capping everything off with John Savill's exam cram course.



The focus of my study covers four main pillars of the exam:

* Describe core data concepts (25–30%)
* Identify considerations for relational data on Azure (20–25%)
* Describe considerations for working with non-relational data on Azure (15–20%)
* Describe an analytics workload on Azure (25–30%)





This is just the beginning of a much larger journey. Once I pass the DP-900, I plan to use this foundational knowledge to specialize further into data engineering/analysis or cloud architecture.
